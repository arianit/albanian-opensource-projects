# Contributing

We welcome contributions! If you'd like to add a project or improve an entry, the short flow below is the recommended way to contribute to this repository.

### Step 1: Find a project to add
- The project should be related to Kosovo, Albania, Albanian language, or Albanian developers.
- It must be publicly available and have an open source license.

### Step 2: Fork and clone
```bash
# Fork this repository on GitHub, then clone your fork
git clone https://github.com/YOUR-USERNAME/albanian-opensource-projects.git
cd albanian-opensource-projects
```

### Step 3: Create a branch
```bash
# Create a descriptive branch name
git checkout -b add-project-name
```

### Step 4: Add your project
- Edit `README.md` and add your project under the appropriate section following the repository's entry format (Title, Description, Repository (https://...), Technology, Status, Maintainer).

### Step 5: Commit and push
```bash
# Stage your changes
git add README.md

# Commit with a clear message
git commit -m "Add [Project Name] to [Section Name]"

# Push to your fork
git push origin add-project-name
```

### Step 6: Create a Pull Request
- Open a Pull Request to the main repository and describe what project you're adding and why it's relevant.

### Alternative: Open an Issue
- If you prefer not to submit a PR, open an issue suggesting the project and include the details in the standard format.

---

For full formatting rules, examples, and quality guidelines, see `README.md`. If you'd like me to add a small PR template or a one-page checklist at the top of this file, tell me and I'll add it.
