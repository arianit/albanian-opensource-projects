# Albanian Open Source Projects

A crowdsourced collection of open source projects related to Kosovo, Albania, Albanian language, and Albanian developers. This repository aims to promote open source culture within the Albanian community, celebrate the contributions of Albanian developers worldwide, and provide a centralized resource for discovering Albanian-related open source projects.

## Language & Linguistics

#### Albanian NLP Library

A comprehensive library designed to enable Natural Language Processing tasks in the Albanian language, including tokenization, stemming, and text processing.

- **Repository:** https://github.com/arditdine/albanian-nlp
- **Technology:** Python, NLP
- **Status:** Active
- **Maintainer:** Ardit Dine

#### NLP for Albanian Language

Project focused on implementing various NLP features and models specifically tailored for the Albanian language, including machine learning approaches.

- **Repository:** https://github.com/thinkgradient/nlp-for-albanian
- **Technology:** Python, Machine Learning
- **Status:** Maintained
- **Maintainer:** ThinkGradient Team

#### GiellaLT Albanian Grammar

Finite state and Constraint Grammar based analyzers, proofing tools, and linguistic resources for Albanian language processing and spell checking.

- **Repository:** https://giellalt.github.io/lang-sqi/
- **Technology:** FST, Constraint Grammar, XML
- **Status:** Active
- **Maintainer:** GiellaLT Infrastructure

#### Albanian Spellchecker Dictionary

GPL-licensed spell checking dictionary for Albanian, compatible with OpenOffice, LibreOffice, and other applications supporting MySpell/Hunspell format.

- **Repository:** https://extensions.openoffice.org/en/project/albanian-spellchecker-dictionary
- **Technology:** MySpell/Hunspell
- **Status:** Active
- **Maintainer:** Albanian Language Community

#### Cyrillic Transliteration

Tool to transliterate text between Cyrillic and Latin scripts, useful for processing multilingual content in Balkan languages including Albanian.

- **Repository:** https://github.com/opendatakosovo/cyrillic-transliteration
- **Technology:** Python
- **Status:** Maintained
- **Maintainer:** Open Data Kosovo

#### Uniparser-Grammar-Albanian

Rule-based morphological analyzer for literary Albanian (with Gheg dialect support) providing full lemmatization, POS tagging, and grammatical tagging via a Python package, built on a 31-million-word corpus.

- **Repository:** https://github.com/timarkh/uniparser-grammar-albanian
- **Technology:** Python, Morphological Analysis
- **Status:** Active
- **Maintainer:** Timofey Arkhangelskiy

#### Albanian-POS Treebank

Manually annotated treebank for Standard Albanian (~118K tokens) with neural morphological tagging and lemmatization models based on the Universal Dependencies schema.

- **Repository:** https://github.com/NeldaKote/Albanian-POS
- **Technology:** NLP, Treebank, Neural Models
- **Status:** Research / In Progress
- **Maintainer:** Nelda Kote

## Web & Mobile Applications

#### Secure Storage Android

Android library for storing credentials and sensitive data securely on Android devices using encryption best practices.

- **Repository:** https://github.com/drilon305/secure-storage-android
- **Technology:** Android, Java/Kotlin
- **Status:** Active
- **Maintainer:** Drilon Reçica

#### SMS Parser Android

Android library for receiving, parsing, and processing SMS messages, enabling developers to build SMS-based features efficiently.

- **Repository:** https://github.com/drilon305/sms-parser-android
- **Technology:** Android, Java/Kotlin
- **Status:** Maintained
- **Maintainer:** Drilon Reçica

#### Open ARBK

Business registration search engine featuring data from over 170,000 businesses in Kosovo, scraped from the Kosovo Business Registration Agency for economic analysis.

- **Repository:** https://github.com/opendatakosovo/open-arbk
- **Technology:** Python, Web Scraping, Data Visualization
- **Status:** Active
- **Maintainer:** Open Data Kosovo

#### Dogana Kosoves

Data analysis and visualization project focused on Kosovo Customs trade balance and economic trends, providing insights into import/export patterns.

- **Repository:** https://github.com/ndriqimlahu/dogana-kosoves
- **Technology:** Python, Data Analysis, Visualization
- **Status:** Active
- **Maintainer:** Ndriqim Lahu

#### 24 Hours of Traffic — Prishtina

Python scripts that generate animated 24-hour traffic visualizations for Prishtina using custom Google Maps styling, providing data-driven insight into urban traffic patterns.

- **Repository:** https://github.com/ArditXhaferi/24-hours-of-traffic-prishtina
- **Technology:** Python, Data Visualization, Google Maps
- **Status:** Active
- **Maintainer:** Ardit Xhaferi

## Tools & Utilities

#### css.gg

Collection of 700+ minimalist, pure CSS, SVG & Figma UI icons available in multiple formats including SVG Sprite, styled-components, NPM package, and API.

- **Repository:** https://github.com/astrit/css.gg
- **Technology:** CSS, SVG, JavaScript
- **Status:** Active - Seeking Contributors
- **Maintainer:** Astrit Malsija

#### WordPressify

Modern build system and workflow automation tool designed to streamline WordPress theme development with Gulp, Webpack, and live reloading.

- **Repository:** https://github.com/luangjokaj/wordpressify
- **Technology:** Node.js, Webpack, Gulp, WordPress
- **Status:** Active
- **Maintainer:** Luan Gjokaj

#### GoPablo

Modern static site generator powered by Webpack, featuring hot module replacement, Sass support, and optimized production builds.

- **Repository:** https://github.com/luangjokaj/gopablo
- **Technology:** Node.js, Webpack, Static Site Generation
- **Status:** Active
- **Maintainer:** Luan Gjokaj

#### FCMBundle

Symfony bundle for sending push notifications to mobile devices through Firebase Cloud Messaging API, simplifying mobile notification integration.

- **Repository:** https://github.com/redjanym/FCMBundle
- **Technology:** PHP, Symfony, Firebase
- **Status:** Maintained
- **Maintainer:** Redjan Ymeraj

#### 5HQ1P — Albanian Programming Language

World's first Albanian-keyword programming language with a Python-based interpreter, allowing developers to write code using Albanian words (e.g. `nese` for `if`), aimed at making programming accessible to Albanian-speaking beginners.

- **Repository:** https://github.com/ArditXhaferi/5HQ1P
- **Technology:** Python, Programming Language, Interpreter
- **Status:** Active
- **Maintainer:** Ardit Xhaferi

#### Albanian Early Bound

XrmToolBox plugin for generating strongly-typed early-bound Dynamics 365/CRM entity classes via a GUI, with granular attribute and relationship filtering to minimize assembly size. Also supports CLI usage after initial model generation.

- **Repository:** https://github.com/albanian-xrm/Early-Bound
- **Technology:** C#, Dynamics 365, Power Platform
- **Status:** Active
- **Maintainer:** Betim Beja (AlbanianXrm)

## Data & Research

#### Albanian Sentiment Analysis Dataset

Human-annotated dataset of 10,132 social media comments in Albanian for sentiment analysis, addressing the scarcity of labeled data for low-resource languages.

- **Repository:** Available through research publications and Papers with Code
- **Technology:** NLP Dataset, Machine Learning
- **Status:** Published Dataset
- **Maintainer:** Research Community

#### CC100-Albanian Dataset

Large-scale Albanian language corpus extracted from Common Crawl for language modeling and NLP research.

- **Repository:** https://huggingface.co/datasets/cc100
- **Technology:** Text Corpus, NLP
- **Status:** Active
- **Maintainer:** Facebook Research / Hugging Face

#### OSCAR Albanian Corpus

Part of the Open Super-large Crawled ALMAnaCH coRpus, providing extensive Albanian language data for machine learning and linguistic research.

- **Repository:** https://oscar-corpus.com/
- **Technology:** Large Text Corpus, NLP
- **Status:** Active
- **Maintainer:** OSCAR Team / Inria

#### Albanian Abusive Language Detection Dataset

Dataset formatted following OffensEval convention for detecting abusive and offensive language in Albanian from social media platforms like Instagram and YouTube.

- **Repository:** Available through research publications
- **Technology:** NLP Dataset, Classification
- **Status:** Published Dataset
- **Maintainer:** Research Community

#### MULTEXT-East Albanian Resources

Multilingual dataset including Albanian language resources with morphosyntactic specifications, lexica, and annotated parallel corpus ("1984").

- **Repository:** http://nl.ijs.si/ME/
- **Technology:** Linguistic Resources, XML
- **Status:** Maintained
- **Maintainer:** MULTEXT-East Consortium

#### Albanian News Articles Dataset

First publicly available Albanian-language news corpus — over 3 million articles from Kosovo news sources (2007–2020) with metadata, designed for topic modeling, category prediction, and NLP research.

- **Repository:** https://www.kaggle.com/datasets/gentrexha/kosovo-news-articles-dataset
- **Technology:** NLP Dataset, Text Corpus
- **Status:** Published Dataset
- **Maintainer:** Gent Rexha, Majlinda Llugiqi

#### Albanian Fake News Corpus

First fake news detection corpus for Albanian — ~4,000 balanced fake and true news articles compiled with a hybrid algorithmic approach, supporting misinformation research in low-resource languages.

- **Repository:** https://www.kaggle.com/datasets/gentrexha/albanian-fake-news-corpus
- **Technology:** NLP Dataset, Classification
- **Status:** Published Dataset
- **Maintainer:** Gent Rexha

## AI Models

#### Bleta-Logjike

Advanced Albanian-capable large language model fine-tuned on Gemma-3 architecture, specialized in multi-step reasoning and mathematical problem-solving for Albanian language.

- **Repository:** https://huggingface.co/Dialogo
- **Technology:** LLM, Gemma-3, PyTorch
- **Status:** Active
- **Maintainer:** Dialogo AI Team

#### Whisper Albanian (Shqip) Large V3 Turbo

Fine-tuned version of OpenAI's Whisper model specifically for Albanian language including Gheg dialect, trained on 200 hours of high-quality Albanian audio.

- **Repository:** https://huggingface.co/Kushtrim/whisper-large-v3-turbo-shqip
- **Technology:** Speech Recognition, Whisper, PyTorch
- **Status:** Active
- **Maintainer:** Kushtrim

#### Albanian BERT

BERT model pre-trained on Masked Language Modeling tasks using Albanian language datasets, enabling transfer learning for downstream NLP tasks.

- **Repository:** Available through Hugging Face — search Albanian language models
- **Technology:** BERT, Transformers, PyTorch
- **Status:** Research / Available
- **Maintainer:** Research Community

#### Albanian Language Models on Hugging Face

Collection of various Albanian language models including translation, text generation, and classification models available through Hugging Face Hub.

- **Repository:** https://huggingface.co/models?language=sq
- **Technology:** Various (Transformers, NLP)
- **Status:** Growing Collection
- **Maintainer:** Various Contributors
