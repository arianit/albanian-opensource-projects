# Albanian Open Source Projects

A crowdsourced collection of open source projects related to Kosovo, Albania, Albanian language, and Albanian developers. This repository aims to promote open source culture within the Albanian community, celebrate the contributions of Albanian developers worldwide, and provide a centralized resource for discovering Albanian-related open source projects.

## Language & Linguistics

### Albanian NLP Library
**Description:** A comprehensive library designed to enable Natural Language Processing tasks in the Albanian language, including tokenization, stemming, and text processing.
**Repository:** [github.com/arditdine/albanian-nlp](https://github.com/arditdine/albanian-nlp)
**Technology:** Python, NLP
**Status:** Active
**Maintainer:** Ardit Dine

### NLP for Albanian Language
**Description:** Project focused on implementing various NLP features and models specifically tailored for the Albanian language, including machine learning approaches.
**Repository:** [github.com/thinkgradient/nlp-for-albanian](https://github.com/thinkgradient/nlp-for-albanian)
**Technology:** Python, Machine Learning
**Status:** Maintained
**Maintainer:** ThinkGradient Team

### GiellaLT Albanian Grammar
**Description:** Finite state and Constraint Grammar based analyzers, proofing tools, and linguistic resources for Albanian language processing and spell checking.
**Repository:** [giellalt.github.io/lang-sqi](https://giellalt.github.io/lang-sqi/)
**Technology:** FST, Constraint Grammar, XML
**Status:** Active
**Maintainer:** GiellaLT Infrastructure

### Albanian Spellchecker Dictionary
**Description:** GPL-licensed spell checking dictionary for Albanian, compatible with OpenOffice, LibreOffice, and other applications supporting MySpell/Hunspell format.
**Repository:** [extensions.openoffice.org/en/project/albanian-spellchecker-dictionary](https://extensions.openoffice.org/en/project/albanian-spellchecker-dictionary) | [sourceforge.net/projects/albdic](https://sourceforge.net/projects/albdic/)
**Technology:** MySpell/Hunspell dictionary format
**Status:** Active
**Maintainer:** Albanian Language Community

### Cyrillic Transliteration
**Description:** Tool to transliterate text between Cyrillic and Latin scripts, useful for processing multilingual content in Balkan languages including Albanian.
**Repository:** [github.com/opendatakosovo/cyrillic-transliteration](https://github.com/opendatakosovo/cyrillic-transliteration)
**Technology:** Python
**Status:** Maintained
**Maintainer:** Open Data Kosovo

## Data & Research

### Albanian Sentiment Analysis Dataset
**Description:** Human-annotated dataset of 10,132 social media comments in Albanian for sentiment analysis, addressing the scarcity of labeled data for low-resource languages.
**Repository:** Available through research publications and Papers with Code
**Technology:** NLP Dataset, Machine Learning
**Status:** Published Dataset
**Maintainer:** Research Community

### CC100-Albanian Dataset
**Description:** Large-scale Albanian language corpus extracted from Common Crawl for language modeling and NLP research.
**Repository:** [huggingface.co/datasets/cc100](https://huggingface.co/datasets/cc100) (Albanian subset)
**Technology:** Text Corpus, NLP
**Status:** Active
**Maintainer:** Facebook Research / Hugging Face

### OSCAR Albanian Corpus
**Description:** Part of the Open Super-large Crawled ALMAnaCH coRpus, providing extensive Albanian language data for machine learning and linguistic research.
**Repository:** [oscar-corpus.com](https://oscar-corpus.com/) | Available through Hugging Face
**Technology:** Large Text Corpus, NLP
**Status:** Active
**Maintainer:** OSCAR Team / Inria

### Albanian Abusive Language Detection Dataset
**Description:** Dataset formatted following OffensEval convention for detecting abusive and offensive language in Albanian from social media platforms like Instagram and YouTube.
**Repository:** Available through research publications
**Technology:** NLP Dataset, Classification
**Status:** Published Dataset
**Maintainer:** Research Community

### MULTEXT-East Albanian Resources
**Description:** Multilingual dataset including Albanian language resources with morphosyntactic specifications, lexica, and annotated parallel corpus ("1984").
**Repository:** [nl.ijs.si/ME](http://nl.ijs.si/ME/)
**Technology:** Linguistic Resources, XML
**Status:** Maintained
**Maintainer:** MULTEXT-East Consortium

### Bleta-Logjike
**Description:** Advanced Albanian-capable large language model fine-tuned on Gemma-3 architecture, specialized in multi-step reasoning and mathematical problem-solving for Albanian language.
**Repository:** [huggingface.co/Dialogo](https://huggingface.co/Dialogo) | [dialogo.chat](https://dialogo.chat/)
**Technology:** LLM, Gemma-3, PyTorch
**Status:** Active
**Maintainer:** Dialogo AI Team

### Whisper Albanian (Shqip) Large V3 Turbo
**Description:** Fine-tuned version of OpenAI's Whisper model specifically for Albanian language including Gheg dialect, trained on 200 hours of high-quality Albanian audio.
**Repository:** [huggingface.co/Kushtrim/whisper-large-v3-turbo-shqip](https://huggingface.co/Kushtrim/whisper-large-v3-turbo-shqip)
**Technology:** Speech Recognition, Whisper, PyTorch
**Status:** Active
**Maintainer:** Kushtrim

### Albanian BERT
**Description:** BERT model pre-trained on Masked Language Modeling tasks using Albanian language datasets, enabling transfer learning for downstream NLP tasks.

**Status:** Research / Available
**Maintainer:** Research Community

## How to Contribute

We welcome contributions from everyone! Here's how you can help expand this repository:

### Step 1: Find a Project to Add
Identify an open source project that meets our criteria:
- Related to Kosovo, Albania, Albanian language, or Albanian developers
- Publicly available with an open source license
- Active or historically significant to the Albanian open source community

### Step 2: Fork and Clone
```bash
# Fork this repository on GitHub, then clone your fork
git clone https://github.com/YOUR-USERNAME/albanian-opensource-projects.git
cd albanian-opensource-projects
```

### Step 3: Create a Branch
```bash
# Create a descriptive branch name
git checkout -b add-project-name
```

### Step 4: Add Your Project
Edit `README.md` and add your project under the appropriate section following the format specified in [CONTRIBUTING.md](CONTRIBUTING.md).

### Step 5: Commit and Push
```bash
# Stage your changes
git add README.md

# Commit with a clear message
git commit -m "Add [Project Name] to [Section Name]"

# Push to your fork
git push origin add-project-name
```

### Step 6: Create a Pull Request
Go to the original repository on GitHub and create a pull request from your branch. Describe what project you're adding and why it's relevant.

### Alternative: Open an Issue
If you're not comfortable with Git, you can simply [open an issue](../../issues) suggesting a project to add, and we'll review it!
